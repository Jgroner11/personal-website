Projects

Selected work, loaded from one YAML file.

Add projects to projects.yaml with a title, local video path, abstract, and GitHub link. The page will render however many entries you include.

YAML structure

projects:
  - title: "Project name"
    video: "assets/videos/project-video.mp4"
    abstract: |
      A short description of the project.
      You can make this multiple lines.
    github: "https://github.com/your-username/repository"
